É um comando para acessar funcionalidades que não eram possíveis até a versão 16.7. Os hooks foram lançados a partir da versão  16.8 e eles liberam o uso de vários recursos de forma simples através de funções. Eles também ajudam a organizar a lógica utilizada dentro dos componentes.

Resumo: É um facilitador no desenvolvimento que serve para deixar os componentes mais simples e permite manipular o código de uma melhor forma.
