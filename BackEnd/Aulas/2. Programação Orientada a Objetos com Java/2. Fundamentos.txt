 Aula - Fundamentos

Orientação a Objetos: É um paradigma de análise, projeto e programação de sistemas de software baseado na composição e interação entre diversas unidades de software chamados de objetivos.

 Fundamentos- 3 Pilares

* Abstação: Processo pelo qual se isolam características de um objeto, considerando os que tenham em comum certos grupos de objetos. Se preocupar com o essencial e que pode ser reutilizado.

* Reuso: Capacidade de criar novas unidades de código a partir de outras já existentes.

* Encapsulamento: Capacidade de esconder complexidades e proteger dados.
