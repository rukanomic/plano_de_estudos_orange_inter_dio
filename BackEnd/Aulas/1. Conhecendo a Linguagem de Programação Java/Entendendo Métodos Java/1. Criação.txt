Aula 01 - Conceituação de Métodos

Método: É uma porção de código (sub-rotina) que é disponibilizada por uma classe. Este é executado quando é feita uma requisição a ele. São responsáveis por definir e realizar um determinado comportamente.

Aula 03 - Conceitos Inerentes aos métodos e boas práticas na sua criação

Assinatura: É a forma de identificar unicamente o método.
Ex: Ass = NOME + PARÂMETROS

Mensagem: É o ato de solicitar ao método que o mesmo execute. Esta pode ser direcionada a um objeto ou a uma classe.

Boas Práticas

* Nomes devem ser descrtivos, mas curtos
* Utilizar Camel Case
* Deve possuir entre 80/120 linhas
* Evite lista de parâmetros longas
* Visibilidades adequadas
